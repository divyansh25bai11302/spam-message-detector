# spam-message-detector
Project Name - Spam Message Detector, Created By - Divyansh Mishra, Institute - VIT BHOPAL

1. What the project does?
   
As the name suggests, the project detects whether a message is spam or not.

2. Why is this project useful?
   
The project is useful because it helps in detecting spam messages easily which will protect the innocent people from scams. The people who can not differentiate between spam and real messages can simply write their messages here and this project will tell whether it is a spam message or a normal message.

3. How can the users get started with this project?

To get started, the user must run the code. As the code runs, it will ask for only one detail, which is ENTER MESSAGE where the user has to type the message in which he/she is confused whether it is spam or not and the program will tell if it is SPAM or HAM ( HAM = not spam).
